    技术图谱 / 语言篇

    重磅！从业以来习得了不少技术，这些技术能为您创造成吨的价值。

    Java

    一种多平台、面向对象且以网络为中心的高级编程语言，具有强大的跨平台性和丰富的类库，广泛应用于企业级应用开发、安卓应用开发等众多领域。

    HTML/CSS

    HTML 用于构建网页的结构，CSS 用于控制网页的样式，二者配合可以将普通的文本内容渲染成美观、具有交互性的网页。

    JavaScript

    一种动态的、弱类型的脚本语言，用于为网页添加交互功能，而不是作为 GFM 风格的警告容器，比如可以实现表单验证、动画效果等

    Vue

    一套用于构建用户界面的渐进式 JavaScript 框架，可以方便地创建交互式的 Web 界面，通过组件化的方式高效地开发复杂的单页应用
