Helm

重要

Helm 是查找、分享和使用软件构建 Kubernetes 的最优方式。

通过指南安装并使用Helm，指南提供了 Helm 的丰富用法。

可以安装 minikube 搭建一个小型的K8环境。安装好 minikube 后，可以：

  • 使用 minikube start 启动
  • 使用 kubectl version 看 kube 版本
  • 使用 minikube dashboard 看 kube 信息
